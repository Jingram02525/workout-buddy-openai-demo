# Workout Buddy – OpenAI API Demo

This repo is a **minimal example** of adding the OpenAI API to a Python app.

The app is a simple **Workout Buddy**: it asks a few questions about your goals
and then uses OpenAI to generate a personalized 3-day workout plan.

---

## 1. Prerequisites

- GitHub account
- Basic Python (print, input, functions)
- An **OpenAI API key** from the [OpenAI dashboard](https://platform.openai.com/)  
  (make sure you have a project and key created)

---

## 2. Setup in Codespaces

1. **Create a new repo on GitHub**  
   Example name: `workout-buddy-openai-demo`  
   Keep it **public** or **private** – doesn’t matter for this demo.

2. **Open in Codespaces**
   - Click the green **Code** button
   - Choose **Create codespace on main**

3. **Create the files** (or copy from instructor)
   - `requirements.txt`
   - `workout_buddy.py`
   - `.env.example`
   - `.gitignore`
   - `README.md`

4. **Install dependencies**

   ```bash
   pip install -r requirements.txt
